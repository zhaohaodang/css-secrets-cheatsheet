`background-position` 属性已经得到扩展，它允许我们指定背景图片距离任意角的偏移量。只要我们在偏移量前面指定关键字（top / right / bottom / left）。

举例来说，如果想让背景图片跟右边缘保持 20px 的偏移量，同时跟底边保持 10px 的偏移量，可以这样做：

```css {5}
div {
  width: 200px;
  height: 200px;
  background: url(...) no-repeat #58a;
  background-position: right 10px bottom 20px;
}
```
效果图：

import myImageUrl from "./img/img1.png";

<div
  style={{
    width: 200,
    height: 200,
    background: `url(${myImageUrl}) no-repeat #58a`,
    backgroundPosition: "right 10px bottom 20px",
  }}
></div>
